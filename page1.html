My Game

Play My Game My Reflection

My game development journey started with an idea : https://docs.google.com/document/d/1AoYkYsDz2H51y1DYqLOP6rhGWD4EyFg7LMvbaCKAhzI/edit?usp=sharing (First Idea)

Then I started.

Eventually i saw that everything was taking too much time so i deleted the game and started from a template but from the amount of time left i tried to add many ideas that first came to my head. This resulted in more time being spent on feature rather than the actual game itself. Then i decided to watch a tutorial. I followed through and decided to use my own sprites. Eventually i made a game with the help of a few tutorials and i hade a few friends to test it before publishing it to the website.

Most Rewarding

The most rewarding part of the development of the game was the final product. The final game that finally works and didnt take me long to make and people seemed to like it and the games met my requirements such as being difficult but not too much but also entertaining. I was expecting a bigger game like those popular games but then i realised that it takes time and money to make games like that. In the end even i found the game fun and its not too repetitive because it eventually gets harder but this is the most rewarding part in my opinion.

Most Challenging

The most challenging part of the development of the game was transforming all ideas into the game which resulted in multiple prototype games. I had many ideas which i wanted to use and that was hard to do because of the limited amount of resources like gdevelop, the software i used for making the games. This resulted in me doing last minutes adjustments only to dump these ideas and start afresh.

If you were to do this again, what would you do differently?

If i were to do things differently, i would have sticked to a simpler idea to save time for later updates as i know myself and i would want to add features before making the actual games. Also using my time efficiently as i know that sometimes i would spend too much time or too less time so that interfered with my schedule. Next time i would either do another skill or use a different software or using my programming skills to make a game myself.

Time management/Goal setting

The time management was one of my issues. Most of the times i would work on the game at home and i would usually do it after doing homework and revising or on a weekend. But sometimes i would get too obsessed and work on it too much which led to many problems like not having enough time for tests or homeworks, or even to be productive and ditching ideas last minute as i didnt have enough time.

I managed to reach my goal in making a game but the journey should have taken less time if it wasnt for my bad time management. Unfortunately i would have added more stuff but atleast i finished it.
